# Unit Test demo using Jest

## Installation

```
> npm install jest --save-dev
> npm i -D @types/jest
> npm install --save-dev @types/express
```

## Configuration

``package.json``

```JSON
"scripts": {
    "test": "jest"
  },
```

## Execution

```
> npm test
```
